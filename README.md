# Online Toolbox

一組純前端、零相依的瀏覽器小工具,透過 GitHub Pages 發佈。

## 工具

| 工具 | 說明 | 線上開啟 |
|---|---|---|
| **Scrapbook** | 貼上即時 render(Markdown / HTML / JSON / CSV / 程式碼 / 圖片)、可存於本機並複製(保留格式) | <https://glennfriend.github.io/online-toolbox-public/scrapbook/> |
| **Calcpad** | numbr 風格逐行筆記型計算機:數學運算式 + 帶時區的日期加減 / 時區轉換,內容即時編進網址可分享 | <https://glennfriend.github.io/online-toolbox-public/calcpad/> |

> 在 GitHub 上看到的是原始碼;請點上面的連結用排版後的網頁操作。
> 各工具細節見各自資料夾的 README(如 [`scrapbook/`](scrapbook/README.md))。
