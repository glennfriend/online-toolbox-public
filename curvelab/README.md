# Curvelab

互動式「公式圖形庫」:選一個公式,拉動參數(或按播放),即時看圖形怎麼變、怎麼被「畫」出來,並解釋每個公式的意義。每個圖**獨立成一個檔案、好加好拆**。純前端、零第三方相依 —— 用瀏覽器開即可,不需要 server。

---

## 目標

把「有意義的公式」做成可以動手玩的圖,讓使用者透過調整參數理解公式,而不只是看靜態圖。重點放在**對國中生有意義、能學、能看出「曲線是怎麼形成的」**的圖。

每個圖獨立成一個模組,互不影響;新增、移除一個圖都不會動到其他圖。

---

## 目前的圖(18 個)

分三類:

### 一、課綱核心(函數與方程式)

| 圖 | 公式 | 學到什麼 |
|---|---|---|
| 直線 | `y = mx + b` | 斜率、y 截距;右移 1 上升 m 的斜率三角形 |
| 二次函數 | `y = ax²+bx+c` | 開口方向與胖瘦、頂點、y 截距 |
| 拋物線(生成) | `y = x²` | 「曲線 = 點的集合」:點沿 x 掃,沿路畫出曲線 |
| 反比 | `y = k/x` | 反比例、xy 為定值、漸近線(碰不到軸) |
| 絕對值 | `y = a\|x−h\|+k` | V 形、頂點平移 |
| 聯立方程式 | 兩線交點 | 聯立方程式的解 = 兩條線的交點;平行=無解 |
| 方程式的解 | `f(x) = 0` | 自由輸入式子,數值找根,標在數線上 |
| 畢氏定理 | `a²+b²=c²` | 三邊正方形面積關係(3-4-5) |

### 二、圓與圓錐曲線

| 圖 | 公式 | 學到什麼 |
|---|---|---|
| 圓 | `x²+y²=r²` | 拖點看座標,x²+y² 恆等於 r² |
| 橢圓 | `x²/a²+y²/b²=1` | 被拉長的圓;兩焦點距離和固定 |
| 雙紐線 | `r²=a²cos2θ` | ∞ 形;到兩定點距離「相乘」固定 |

### 三、生成型(看點怎麼描出曲線,按播放動畫)

| 圖 | 形成方式 |
|---|---|
| 正弦曲線 `y=sin(x)` | 點繞單位圓轉,它的高度就是 sin → 旋轉變成波 |
| 懸鏈線 `y=a cosh(x/a)` | 鏈條/電線自然下垂的形狀(像拋物線卻不是) |
| 擺線 | 滾動輪子,輪緣上一點的軌跡 |
| 螺線 `r=bθ` | 一邊等速轉、一邊等速往外 → 等間距螺旋 |
| 心臟線 `r=a(1−cosθ)` | 極座標半徑隨角度伸縮,描出愛心 |
| 星形線 `x=a cos³t` | 小圓在大圓內滾動,圓上一點的軌跡 |
| 利薩茹 | 左右、上下兩個擺動合成,頻率比 p:q 決定花樣 |

## 架構

刻意不用框架、不用 ES modules(用傳統 `<script>`,所以**直接用瀏覽器開單一 `index.html` 即可,連 server 都不需要**)。核心與「圖形功能」分離,圖可插拔。

```
curvelab/
├── index.html              版面 + 樣式 + 依序載入下列 JS
├── js/
│   ├── theme.js            共用:主題色(從 CSS 變數讀出給 SVG 用)
│   ├── plot.js             共用:createPlot() 繪圖工具(座標換算、格線/軸、函數/參數取樣)
│   ├── ui.js               共用:UI 小工具 + tracer(生成型動畫的共用描繪器)
│   ├── expr.js             共用:運算式解析器(無 eval)+ 數值找根
│   ├── registry.js         核心:registerGraph 登記表 + App 外殼(切換、掛載、收尾)
│   └── graphs/             每個圖一個獨立檔案
│       ├── line.js  quadratic.js  parabola.js  inverse.js  abs.js
│       ├── circle.js  ellipse.js  lemniscate.js
│       ├── system.js  roots.js  pythagoras.js
│       └── sine.js  catenary.js  cycloid.js  spiral.js  cardioid.js  astroid.js  lissajous.js
└── README.md
```

載入順序:共用 library → 各圖模組(此順序 = 右側按鈕順序)。各圖在載入時呼叫全域的 `registerGraph(...)` 把自己登記進 `GRAPHS`;`App` 在 `DOMContentLoaded` 時生成按鈕、掛載第一個圖。

### 圖形模組介面

```js
registerGraph({
  id:   'circle',          // 唯一識別
  name: '圓',              // 按鈕顯示名稱
  eq:   'x²+y²=r²',        // 按鈕副標(公式)
  create(root){            // 把自己的 UI 與邏輯全建進 root,狀態關在自己的閉包裡
    // ... 用 createPlot / UI /(生成型可用 UI.tracer)
    return { destroy(){} }; // 選用:切換離開時收尾(例如停掉動畫)
  },
});
```

切換圖時,`App` 會先呼叫上一個圖的 `destroy()`(若有),清空顯示區,再建立新圖,所以圖與圖之間不會互相污染。

**新增一個圖**(兩步):
1. 在 `js/graphs/` 新增一個檔案,裡面 `registerGraph({...})`。
2. 在 `index.html` 的圖模組清單多加一行 `<script src="js/graphs/你的檔.js"></script>`。

核心(共用 library)完全不用動。**移除一個圖** = 刪檔 + 刪那行 `<script>`。

### 共用繪圖工具 `createPlot()`

`plot.js` 提供座標換算 `sx/sy`、`grid()`、`axes()`、把函數取樣成路徑的 `funcPath()`、把參數式取樣成路徑的 `paramPath()`、以及拖曳用的 `clientToMath()`。

> 想讓圓/橢圓等形狀**不變形**,讓 `width/height` 比例 = `x/y` 範圍比例(兩軸每單位像素相等)。寬而扁的圖(正弦、擺線)就是這樣設定的。

### 生成型動畫 `UI.tracer()`

`ui.js` 的 `UI.tracer(plot, opts)` 把「按播放/拖 t,點沿參數曲線 (fx(t), fy(t)) 移動、沿路把曲線畫出來」這件事做成共用元件;各生成型的圖(正弦、擺線、螺線、心臟線、星形線…)共用它,再用 `opts.decorate` 加上自己的招牌示意(旋轉半徑、滾動的圓…)。

### 運算式解析器(圖:方程式的解)

`expr.js` 自寫一個小型解析器(`tokenize → shunting-yard → RPN → eval`),**不使用 `eval()`**,只做數學求值。支援運算子 `+ - * / ^`、括號、一元負號、隱含乘法(`5x`、`2sin(x)`、`(x-1)(x-2)`)、函數 `sin cos tan sqrt abs exp ln log`、常數 `pi e`。找根用數值法(密集取樣找變號區間 → 二分逼近 → 去重 → 漂亮數字)。

## 已知取捨

- 找根範圍預設 `-30 ~ 30`、步進 `0.02`;範圍外或彼此非常靠近的根可能漏抓。
- 各圖可視範圍固定;超出範圍的部分會被裁切。
- 圓的 `x`、`y` 不開放滑桿調(它們是「點的座標」而非參數),只有半徑 `r` 可調——刻意的教學設計。
- 生成型的「漂亮形狀」(心臟線、星形線、利薩茹等)雖超出國中課綱,但用來示範「曲線 = 移動點的軌跡」非常直觀,故保留。
- 配色用 CSS 變數 + `prefers-color-scheme`,自動跟隨系統深淺色;SVG 顏色於載入時從 CSS 變數讀出。

## 本機開發

單一頁面,**直接用瀏覽器開 `index.html` 即可**(傳統 `<script>`,不需要 server)。

若想用 server(任一靜態伺服器皆可):

```bash
cd curvelab && python3 -m http.server 8010   # 開 http://127.0.0.1:8010/
```
