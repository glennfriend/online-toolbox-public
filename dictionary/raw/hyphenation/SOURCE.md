# 連字符樣式 — en_US (Liang / TeX patterns)

- **是什麼**:英文(美式)斷字樣式,Hunspell/LibreOffice 用的 `hyph_en_US.dic`,源自 TeX 的 Liang 演算法樣式。
- **能解鎖**:單字的音節斷點(顯示 hy-phen-a-tion 怎麼斷)。
- **來源**:https://github.com/LibreOffice/dictionaries(`en/hyph_en_US.dic`)
- **授權**:見 LibreOffice dictionaries(TeX 連字樣式,寬鬆)。使用前再確認該檔頭授權。
- **下載日**:2026-06-20

## 檔案
- `hyph_en_US.dic` — 11,130 行樣式(首行為編碼宣告)。104K。

## 重新下載
```bash
curl -L -o hyph_en_US.dic https://raw.githubusercontent.com/LibreOffice/dictionaries/master/en/hyph_en_US.dic
```
