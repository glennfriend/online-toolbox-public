# Webster's Unabridged Dictionary (1913)

- **是什麼**:1913 年版韋氏大字典,公共領域;這份是整理成單一 JSON(word → 定義文字)。
- **能解鎖**:完整、典雅的定義(老式英文,適合做「正式定義」來源)。
- **來源**:https://github.com/matthewreagan/WebstersEnglishDictionary(`dictionary.json`)
- **授權**:原典公共領域;repo 程式 MIT。
- **下載日**:2026-06-20

## 檔案
- `dictionary.json` — 22M,key 為單字(多為大寫),值為定義字串。

## 重新下載
```bash
curl -L -o dictionary.json https://raw.githubusercontent.com/matthewreagan/WebstersEnglishDictionary/master/dictionary.json
```

> 備註:1913 年用語偏古,現代詞彙缺;適合與 WordNet/Wordset 互補。
