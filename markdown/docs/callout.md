# 🔔 Callout 容器

用 `:::` 包起來,做出有顏色的提示框。支援 note / tip / warning / danger / info。
標題可省略(用預設字),也可自己寫:`::: warning 我的標題`。

::: note
一般說明。藍色資訊框。
:::

::: tip 小撇步
記得先暖機模型,第二次就秒載。
:::

::: warning 注意
法務簽核通常很慢,Phase 1 開工就要送 review。
:::

::: danger 危險
這個操作不可逆,請先備份。
:::

::: info
裡面照樣可以放 **粗體**、`程式碼`、清單:
- 第一點
- 第二點
:::

> 跟引用 `>` 的差別:callout 是**多種顏色、有語意**的方框,適合標注意 / 提示 / 警告。
