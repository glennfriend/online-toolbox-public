# 📈 Chart 圖表

在 `chart` 程式碼區塊裡寫「設定 + CSV」,直接渲染成**設定好的圖**(`type` 寫死,一進來就是該圖,不用切換)。引擎用 ECharts。

設定行(開頭):`type:`(必填,radar / bar / line / pie)、`max:`(雷達上限,選填)、`title:`(選填)。
資料:CSV,第一列欄名(第一欄=組名,其餘=各指標/數列),之後每列一組。

## radar:每組一張、由左到右並排

```chart
type: radar
max: 100
角色, 攻擊, 防禦, 速度, 魔法, 體力, 智力
戰士, 90, 85, 40, 20, 95, 30
法師, 40, 30, 60, 95, 50, 90
盜賊, 60, 50, 95, 40, 70, 55
```

## bar:一張圖、各欄為數列

```chart
type: bar
title: 月度收支
月份, 收入, 支出
1月, 100, 80
2月, 120, 90
3月, 95, 110
```

## line:折線

```chart
type: line
週次, 訪客
第1週, 120
第2週, 180
第3週, 150
第4週, 240
```

## pie:第一欄標籤、第二欄值

```chart
type: pie
title: 流量來源
來源, 佔比
直接, 45
搜尋, 30
社群, 25
```
