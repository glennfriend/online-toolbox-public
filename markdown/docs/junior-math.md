# ∑ 國中數學公式

臺灣國中(七、八、九年級)會考常用公式速查。符號:$r$ 半徑、$h$ 高、$b$ 底、$\theta$ 角度、$\pi \approx 3.14$。

## 乘法公式

| 名稱 | 公式 |
|---|---|
| 和的平方 | $(a+b)^2 = a^2 + 2ab + b^2$ |
| 差的平方 | $(a-b)^2 = a^2 - 2ab + b^2$ |
| 平方差 | $(a+b)(a-b) = a^2 - b^2$ |
| 十字交乘 | $(x+a)(x+b) = x^2 + (a+b)x + ab$ |

## 指數律

| 名稱 | 公式 |
|---|---|
| 同底相乘 | $a^m \cdot a^n = a^{m+n}$ |
| 同底相除 | $a^m \div a^n = a^{m-n}$ |
| 次方的次方 | $(a^m)^n = a^{mn}$ |
| 積的次方 | $(ab)^n = a^n b^n$ |
| 零次方 | $a^0 = 1\ (a \ne 0)$ |
| 負指數 | $a^{-n} = \dfrac{1}{a^n}$ |

## 平方根與畢氏定理

| 名稱 | 公式 |
|---|---|
| 根號乘法 | $\sqrt{a}\,\sqrt{b} = \sqrt{ab}$ |
| 根號除法 | $\dfrac{\sqrt{a}}{\sqrt{b}} = \sqrt{\dfrac{a}{b}}$ |
| 畢氏定理 | $a^2 + b^2 = c^2$($c$ 為斜邊) |

## 一元二次方程式 $ax^2+bx+c=0$

| 名稱 | 公式 |
|---|---|
| 公式解 | $x = \dfrac{-b \pm \sqrt{b^2 - 4ac}}{2a}$ |
| 判別式 | $D = b^2 - 4ac$($>0$ 兩相異根、$=0$ 重根、$<0$ 無實根) |
| 根與係數 | $x_1 + x_2 = -\dfrac{b}{a},\ \ x_1 x_2 = \dfrac{c}{a}$ |

## 坐標與一次函數

| 名稱 | 公式 |
|---|---|
| 一次函數 | $y = ax + b$(斜率 $a$、$y$ 截距 $b$) |
| 斜率 | $a = \dfrac{y_2 - y_1}{x_2 - x_1}$ |
| 兩點距離 | $\sqrt{(x_2-x_1)^2 + (y_2-y_1)^2}$ |
| 中點 | $\left(\dfrac{x_1+x_2}{2},\ \dfrac{y_1+y_2}{2}\right)$ |

## 二次函數 $y = ax^2+bx+c$

| 名稱 | 公式 |
|---|---|
| 對稱軸 | $x = -\dfrac{b}{2a}$ |
| 頂點 | $\left(-\dfrac{b}{2a},\ \dfrac{4ac-b^2}{4a}\right)$ |
| 配方式 | $y = a(x-h)^2 + k$,頂點 $(h,k)$ |

## 周長與面積

| 圖形 | 公式 |
|---|---|
| 三角形面積 | $\dfrac{1}{2}bh$ |
| 平行四邊形 | $bh$ |
| 梯形面積 | $\dfrac{1}{2}(a+b)h$($a,b$ 為上下底) |
| 圓周長 | $2\pi r$ |
| 圓面積 | $\pi r^2$ |
| 扇形弧長 | $2\pi r \cdot \dfrac{\theta}{360}$ |
| 扇形面積 | $\pi r^2 \cdot \dfrac{\theta}{360}$ |

## 表面積與體積

| 立體 | 公式 |
|---|---|
| 柱體積 | $(\text{底面積}) \times h$ |
| 錐體積 | $\dfrac{1}{3}(\text{底面積}) \times h$ |
| 圓柱表面積 | $2\pi r^2 + 2\pi r h$ |
| 球體積 | $\dfrac{4}{3}\pi r^3$ |
| 球表面積 | $4\pi r^2$ |

## 相似與比例

| 名稱 | 公式 |
|---|---|
| 相似面積比 | $\left(\dfrac{a_1}{a_2}\right)^2$(邊長比的平方) |
| 相似體積比 | $\left(\dfrac{a_1}{a_2}\right)^3$(邊長比的立方) |

## 統計與機率

| 名稱 | 公式 |
|---|---|
| 平均數 | $\bar{x} = \dfrac{x_1 + x_2 + \cdots + x_n}{n}$ |
| 機率 | $P = \dfrac{\text{符合的情況數}}{\text{所有可能數}}$ |

## 科學記號

| 名稱 | 公式 |
|---|---|
| 表示法 | $a \times 10^n\ (1 \le a < 10)$ |
